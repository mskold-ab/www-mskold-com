# Introduction 
Measuring time is essenntial to many KPIs and dashboards. In most scenarios we asume the unit of time is Busiess hours, andy yet its very hard to build dashboards that actually measure and reflects the true buisness time. 

![alt text](image-1.png)

With Query tile PRO's paid feature Business Hours you can :
+ Calculate the number of business hours between two points in time. 
+ Define you own business hours and business days



# An example 
Lets say something arrives on friday afternon 17:30 , and you start working on it Monday morning 09:30. Would you want that to appear as untouched for 65 hours. 
Using the standard date time subtraction will output 64 hours. 

To get the number of buisess hours between two dayes, just configure it to use our BusinessHours function 

![alt text](image-2.png)

# Defining you working hours
Exactly how the calculation is done is controlled by the Query Tile PRO settings for Business hours.
You can :
+ define the week days counted as business days
+ define the start and stop for business hours
+ add (or import) holidays to the calender
