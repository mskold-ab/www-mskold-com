## Introduction

Enhanced Export PRO makes it easy to generate professional requirements documentation from your project data. Whether you need documentation for stakeholders, developers, auditors, or customers, the export feature allows you to create comprehensive, structured documents in just a few clicks.

## Benefits

Using Enhanced Export PRO to export requirements documentation helps you:

* Create consistent, professional documentation.
* Save time compared to manual document creation.
* Share requirements with stakeholders in a standardized format.
* Maintain traceability between requirements and project artifacts.
* Generate documents suitable for reviews, approvals, and audits.

### Step 1: Select or create a query matching the requirements you want to export
In many cases you would like to export an entire requirement structure with Epics,Features and stories. 
To do so, simply create a tree query starting with epics and include childrens. 

### Step 2: Run the export 
Simply click on the export tab and select the Requirement Specification template. 


## Test case as part of the requirement specification 
If you view your test cases as part of the requirement, you can choose to inlclude both an overview of the test cases,and even expand the test case and show the test steps.
To do so -
1. Click on options to open the options form
2. Check **Include Work items links** and **Expand Work item links** options and select the Tested By relation for the Expand work item links filter

   
## Test results as part of the requirement specification 
If you would like to include proof of verifictaiton, you can choose to inlcude test results as part of the export.

1. Follow the steps to include test cases in the requirment specification.
2. Click on options to open the options form
3. Check the Include test results option
4. If needed you can choose the test plan and configs you want to include by 
5. Selecting test plans for the Test plan filters option
6. Selecting configurations for the Configurations filter options
